# Urusmarkah Pelanggan
Satu komponen utama sistem Urusmarkah. Komponen ini digunakan untuk menyediakan antara muka yang dinamik, interaktif dan mesra pengurus markah. Komponen ini dilengkapi dengan fungsi-fungsi asas untuk mengurus markah peserta pertandingan. Komponen ini juga mengandungi alatan-alatan (_tools_) yang boleh digunakan oleh pengurus markah profesional untuk mendapatkan gambaran keseluruhan data dan mengendali perkara tambahan mengenai pertandingan.
